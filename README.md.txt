# DEAP Emotion Detection App

Multimodal emotion detection using EEG and video data.

## Models
- Video Only (Within + LOSO)
- EEG Only (Within + LOSO)  
- Early Fusion (Within + LOSO)
- Mid Fusion (Within + LOSO)
- Late Fusion (Within + LOSO)

## Run locally
pip install -r requirements.txt
streamlit run app.py
